## 1. 測試目的

前期消融實驗（`mr_ablation_final_report_v1.md`）確認了 CVD 相關訊號在短樣本（2 個月 Tick 資料）下的潛力。本輪測試目標：

1. 在 **28 個月樣本外 KLINE 資料**上驗證訊號組合的穩健性，排除短樣本過擬合風險
2. 比較三個 Enhancer 組合（Baseline / A_loose / A_loose+C），確立最優組態
3. 量化 CVD z-score 過濾器的邊際貢獻

## 4. 核心發現

### 4.1 Baseline：下影線訊號本身無 Alpha

Baseline 跑出 **PF 0.775、MDD 79.9%、淨損 -7,104 USDT**，且手續費總計 10,586 USDT 已超過毛利。這揭示了一個根本問題：`ReversalBarUp` 訊號在無過濾條件下，在 28 個月的完整週期中沒有統計優勢，最大連敗 15 筆。

### 4.2 P2（A_loose）：形態放寬有效但不足

將 `reversal_bar_min_wick` 從 0.5 降至 0.35 後，交易數從 362 驟降至 67（-81.5%）。這反映出更嚴格的形態篩選大幅削減了雜訊訊號：

- MDD 從 79.9% 壓縮至 **24.0%**（改善 -55.9pp）
- 均盈/均損比從 1.26 提升至 **1.32**
- PF 從 0.775 升至 0.944，**但仍< 1.0，策略整體仍虧損**

### 4.3 P1b（A_loose+C）：CVD z-score 是翻轉 Edge 的關鍵

在 A_loose 基礎上加入 `buy_vol_zscore_min=1.0`（要求買方 CVD z-score ≥ 1.0）後：

- 交易數進一步收斂至 31 筆（僅保留原始訊號的 **8.6%**）
- **PF 突破 1.0 達到 1.130**，策略首度轉正
- MDD 壓縮至 **12.7%**，Sharpe 達 **1.61**
- 均盈（347.84）/ 均損（253.40）= **1.37**，盈虧比也改善

CVD z-score 過濾的實質意義：在下影線形成的同時，要求買方累積成交量（CVD）偏離滾動均值 ≥ 1 個標準差。這確認了確實有大量主動買單在支撐該價位，而非隨機蠟燭形態。

## 5. 結論

```
LowerWick 訊號（單獨）      = 雜訊，無 alpha
LowerWick + wick≤0.35       = 大幅減少雜訊，但邊際仍為負
LowerWick + wick≤0.35 + CVD z≥1.0 = 有效 alpha，通過 28 個月 OOS 驗證
```

**P1b 為本輪測試的最優組態**，是唯一在 28 個月樣本外資料上達到正 PF 且具備可接受 MDD 的版本。PF 提升量 +0.355、淨損益改善 +7,666 USDT（相對 Baseline）。

## 6. 下一步行動建議

1. **CVD z 門檻敏感性掃描**
   測試 `buy_vol_zscore_min` = 1.2 / 1.5 / 2.0，觀察 Sharpe / MDD 的改善空間與交易頻率的取捨。

2. **跨品種泛化驗證**
   將 P1b 參數（wick=0.35, z=1.0）直接套到 ETHUSDT，不做任何再優化，確認 alpha 的跨市場普遍性。

3. **Walk-Forward 分析**
   以 6 個月為一段（共 4~5 段），檢驗不同市場環境（牛市/橫盤/熊市）下 PF 的分佈，排除特定時段集中獲利的偽穩健性。

4. **Short 方向擴展**
   本輪測試全為 Long-only（362/31 筆皆為做多）。考慮加入對稱的 Short 訊號以提升交易頻率並改善夏普比。
